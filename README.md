# StockGame

Browser-based wealth-building sim where the player starts at 18, fresh out of high school, and has to grind through jobs, education, banking, investing, property, and business ownership.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## GitHub Pages

This repository is configured to deploy automatically to GitHub Pages from `master` using GitHub Actions.

1. Open the repository on GitHub.
2. Go to `Settings > Pages`.
3. Set `Source` to `GitHub Actions`.
4. Push to `master`.

The Vite base path is already configured for:

`https://prebenjor.github.io/StockGame/`
